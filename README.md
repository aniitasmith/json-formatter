# JSON Formatter

A clean and simple tool to format, validate, and minify JSON with syntax highlighting.

## Features

- **Format** - Beautify JSON with proper indentation
- **Minify** - Compress JSON to a single line
- **Validate** - Real-time error detection with helpful messages
- **Copy** - One-click copy to clipboard
- **Dark Mode** - Toggle between light and dark themes

## Tech Stack

- Next.js 16
- TypeScript
- Tailwind CSS
- React Syntax Highlighter

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
